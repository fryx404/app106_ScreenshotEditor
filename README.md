# 🎨 スクリーンショット エディタ

## 概要
**Snipping Tool** で撮影したスクリーンショットを編集する専門的なエディタです。クリップボードから直接貼り付けて、描画やテキスト追加で注釈やマークアップを簡単に行えます。

## 🚀 **v3.0 - エディタ特化版**
**スクリーンショット撮影機能を削除し、編集に特化した純粋なエディタに生まれ変わりました**

### 💡 **新しいコンセプト**
- **撮影**: Windows Snipping Tool / macOS スクリーンショット機能
- **編集**: このエディタでマークアップ・注釈追加
- **保存**: 編集済み画像をPNG出力

### 🎯 **v3.0の特徴**
- ✨ **クリップボード連携**: Ctrl+V で瞬時に貼り付け
- 🎨 **専門的な編集機能**: 描画・テキスト・移動ツール
- 📋 **ワンクリック貼り付け**: ボタン一つで簡単操作
- 🖱️ **ドラッグ&ドロップ対応**: 画像ファイルを直接ドロップ

## 🚀 主要機能

### 📋 簡単画像読み込み
- **クリップボード貼り付け**: Ctrl+V または「📋 ペースト」ボタン
- **ファイル選択**: 「📁 ファイル選択」でローカル画像を読み込み
- **ドラッグ&ドロップ**: 画像ファイルを直接キャンバスにドロップ
- **自動リサイズ**: 画像サイズに合わせて自動調整

### ✏️ 自由描画機能
- マウスでフリーハンド描画
- カラーピッカーでの色選択
- 線の太さ調整（1-20px）
- スムーズな描画体験とリアルタイムプレビュー

### 📝 高機能テキスト編集
- **任意の場所への配置**: クリックした位置にテキストを挿入
- **豊富なフォント設定**: 
  - 8種類のフォントファミリー（Arial, Helvetica, Times New Roman, Georgia, Verdana, Comic Sans MS, Impact, Courier New）
  - フォントサイズ（8-200px、デフォルト100px）
  - 太字・斜体スタイル
  - カラーピッカーでの色選択
- **テキストオブジェクト管理**: 
  - 追加後のドラッグ&ドロップ移動
  - 個別選択・削除（Delete キー）
  - 背景付きテキストで視認性向上

### ⏮️ アンドゥ/リドゥ機能
- **Photoshop風の履歴管理**: 最大50ステップまでの操作履歴
- **キーボードショートカット**: 
  - Ctrl+Z: 元に戻す
  - Ctrl+Y / Ctrl+Shift+Z: やり直し
- 描画・テキスト追加・移動・削除のすべての操作に対応

### 🎯 多機能ツールバー
- **ツール選択**: 描画・テキスト・移動モードの切り替え
- **リアルタイム設定**: 値の変更が即座に反映
- **グループ化されたUI**: 機能ごとに色分けされた直感的なインターフェース

### 🧹 クリア機能
- **描画クリア**: 描画のみをクリア（テキストは保持）
- **全てクリア**: 描画・テキスト・背景をすべて削除

### 💾 保存・読み込み機能
- 編集済み画像をPNG形式でダウンロード
- ローカル画像ファイルの読み込み
- 自動ファイル名生成（タイムスタンプ付き）

## 🐛 **トラブルシューティング**

### 基本的な確認事項
1. **ページを完全にリフレッシュ**: `Ctrl+F5` (Windows) / `Cmd+Shift+R` (Mac)
2. **ブラウザの開発者ツール**: `F12` → `Console` タブでエラーがないか確認
3. **対応ブラウザ**: Chrome 72+, Firefox 66+, Edge 79+, Safari 13+

### よくある問題と解決法
- **ボタンが反応しない** → ページをリフレッシュして再度お試しください
- **貼り付けができない** → クリップボード権限を許可、またはCtrl+Vをお試しください
- **画像が読み込めない** → 対応形式（PNG, JPG, GIF）かご確認ください
- **Snipping Toolが見つからない** → Windows検索で「Snipping Tool」を検索してください

## 📖 使用方法

### 基本的な流れ
1. **Snipping Tool で撮影**: Windows標準のSnipping Toolでスクリーンショット撮影
2. **エディタで編集**: このアプリで Ctrl+V で貼り付け
3. **マークアップ**: 描画やテキストで注釈追加
4. **保存**: 完成した画像をPNG形式でダウンロード

### 📋 画像の読み込み方法

#### 方法1: クリップボードから貼り付け（推奨）
1. **Snipping Tool** でスクリーンショット撮影（自動でクリップボードにコピー）
2. **エディタで** `Ctrl+V` キーまたは「📋 ペースト」ボタンをクリック
3. **瞬時に読み込み完了**

#### 方法2: ファイルから読み込み
1. **「📁 ファイル選択」** ボタンをクリック
2. **PNG, JPG, GIF** などの画像ファイルを選択
3. **読み込み完了**

#### 方法3: ドラッグ&ドロップ
1. **エクスプローラーから画像ファイル** を選択
2. **キャンバス上にドラッグ&ドロップ**
3. **自動で読み込み**

### 🎨 描画作業
1. **描画ツール**: 「✏️ 描画」を選択
2. **設定調整**: 色と線の太さを調整
3. **描画実行**: マウスドラッグでフリーハンド描画

### ✍️ テキスト編集
1. **テキストツール**: 「📝 テキスト」を選択
2. **フォント設定**: 色・サイズ・フォントファミリー・太字/斜体を設定
3. **テキスト入力**: 入力欄にテキストを入力
4. **配置**: キャンバス上の任意の場所をクリックして配置

### 🤚 テキスト移動
1. **移動ツール**: 「🤚 移動」を選択
2. **選択**: 移動したいテキストをクリック
3. **ドラッグ**: テキストをドラッグして新しい位置に移動
4. **削除**: 選択状態で Delete キーを押すと削除

### ⏮️ 履歴操作
- **元に戻す**: Ctrl+Z または 「↶ 元に戻す」ボタン
- **やり直し**: Ctrl+Y または 「↷ やり直し」ボタン
- 最大50回までの操作を履歴として保存

### 🧹 クリア機能
- **全てクリア**: 「🗑️ クリア」で描画・テキスト・背景をすべて削除

### 💾 保存・読み込み
- **画像保存**: 「💾 保存」ボタンで PNG ファイルとしてダウンロード
- **画像読み込み**: 「📁 画像をアップロード」でローカルファイルを読み込み

## ⌨️ キーボードショートカット

### 基本操作
- **Ctrl+V**: クリップボードから貼り付け（最重要）
- **Ctrl+Z**: 元に戻す（アンドゥ）
- **Ctrl+Y**: やり直し（リドゥ）
- **Ctrl+Shift+Z**: やり直し（リドゥ）
- **Delete**: 選択中のテキストオブジェクトを削除
- **Esc**: 描画ツールに切り替え / 範囲切り取りをキャンセル

### 範囲切り取り時
- **マウスドラッグ**: 切り取り範囲を選択
- **確認ダイアログ**: 「OK」で切り取り確定、「キャンセル」で中止
- **Esc**: 範囲選択をキャンセル

### デバッグ用
- **F12**: ブラウザの開発者ツールを開いてConsoleタブを確認
- **`window.debugEditor()`**: コンソールでアプリの状態を確認

## 注意事項

1. **プライバシー**
   - すべての画像処理はローカルで実行
   - データはサーバーに送信されません
   - 安全にご使用いただけます

2. **ブラウザ権限**
   - クリップボードアクセス時に権限の許可が必要
   - 初回使用時にブラウザが許可ダイアログを表示

3. **パフォーマンス**
   - 大きな画像では動作が重くなる場合があります
   - 推奨解像度: 1920x1080 以下
   - テキストオブジェクトが多い場合は動作が重くなることがあります

4. **使用上のヒント**
   - 細かい編集には適度にズームを活用
   - 大きなテキストサイズ（100px）で視認性向上
   - こまめに保存して作業内容を保護

## トラブルシューティング

### クリップボード貼り付けができない
- ブラウザの権限設定でクリップボードアクセスを許可
- 代わりに「📁 ファイル選択」機能をご利用ください
- Ctrl+V キーを再度お試しください

### 画像が読み込めない
- 対応形式（PNG, JPG, GIF, BMP）であることを確認
- ファイルサイズが過大でないか確認（推奨: 10MB以下）
- ブラウザを再読み込みしてから再度お試しください

### 描画が反応しない
- ツールが正しく選択されているか確認
- Escキーを押してツールをリセット

### Snipping Tool が見つからない（Windows）
- Windows検索で「Snipping Tool」と入力
- Windows 10/11: 「切り取り&スケッチ」も利用可能
- ショートカット: Windows + Shift + S
